The Connected Stack
Five layers. One engine.
Built on infrastructure you already own.
Each layer activates the one above it. Nothing is net-new infrastructure.
5 layers
Xero Decisioning Engine — powered by Salesforce
01
Data Cloud
Unified Customer Profile
Ingests Xero's event stream — invoicing, payroll, bank feed, feature usage — and resolves it into a single real-time profile per customer. This is the source of truth everything else reads from.
02
Einstein Studio
Propensity Scoring
Custom ML models score every customer for churn risk, upsell readiness, and feature adoption gaps — refreshed on each new event. Sub-100ms inference means the score is live when the customer is live.
03
Flow + Decisions
Orchestration Engine
Business rules sit here — offer eligibility, suppression logic, journey entry/exit. Non-technical teams can own this layer without engineering involvement.
04
Marketing Cloud
Email · Push · In-App
Executes the triggered communication — email, push notification, or in-product card — at the moment the decision fires. One canvas controls all three channels from a single trigger.
05
Sales Cloud
CSM Prioritisation
Einstein scores flow directly into CSM task queues. At-risk accounts surface automatically with context on why they're at risk. No manual list-building. No missed signals.
